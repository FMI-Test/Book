## Publication channels checklist

Use one evidence trail for all output formats:

- **Book / hardcopy**
- **Newsletter**
- **Blog / website article**
- **Social media history thread / profile post**
- **Public figure summary page**

For each format, keep:

1. verified name spelling;
2. source citation;
3. archive credit line;
4. image permission status;
5. date last checked.
